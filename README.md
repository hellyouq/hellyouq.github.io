HellPak - Bio

Персональная страница-визитка с атмосферным дизайном и музыкальным плеером.



О проекте

Минималистичный сайт в светло-черно-синей стилистике с эффектом glass



Функции

Click to Enter — стартовый экран с размытием

Динамический заголовок — название вкладки мерцает случайными символами

Блок с контактами — имя, возраст, Discord, Telegram, YouTube

Встроенный плеер — автоматическое воспроизведение трека после входа


Структура файлов

text

assets/

├── fonts/          # Шрифты (Consolas, Unbounded, Radware)

├── images/         # Аватарка (avatar.jpg)

├── sound/          # Музыка (music.mp3)

└── videos/         # Фоновое видео (backmenu.mp4)

Быстрая настройка

Добавь свои файлы в папку assets

При необходимости замени ссылки в блоке .bio-lines

Открой index.html в браузере
